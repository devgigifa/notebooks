Seção 1
Variáveis
Uma variável é um nome que guarda um valor. Em JavaScript moderno usamos let para valores que mudam e const para valores fixos. O antigo var ainda existe, mas tem comportamentos confusos — evite.
let existe só dentro do bloco {} em que foi criada. var vaza para fora do bloco — esse é o problema histórico dele.

Tipos de dados
JavaScript tem tipos primitivos: string (texto), number (número), boolean (verdadeiro/falso), null e undefined. O operador typeof revela o tipo de qualquer valor.

Operadores
Operadores fazem contas, comparam valores e combinam condições. Use sempre === (igual exato) em vez de == — o duplo igual faz conversão de tipo automática e pode surpreender.

Condicionais
Com if / else if / else o programa toma decisões baseadas em condições. O operador ternário (condição ? sim : não) é um if/else compacto para casos simples.

Seção 5
Laços (loops)
Laços repetem código enquanto uma condição for verdadeira. O for é o mais comum: define início, condição de parada e incremento. O while é mais simples quando não sabe quantas repetições terá.

Seção 6
Funções
Funções empacotam código reutilizável. Recebem parâmetros, executam lógica e return devolve o resultado. Arrow functions (=>) são uma sintaxe mais curta, muito usada no JS moderno.
function, arrow function, parâmetros padrão
Seção 7
Arrays
Arrays guardam listas de valores. Índices começam em 0. Os métodos push, pop, splice, sort e reverse manipulam a lista. filter, map e reduce são os métodos modernos mais poderosos.
