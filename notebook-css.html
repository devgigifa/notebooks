O que é CSS?

CSS significa Cascading Style Sheets — folhas de estilo em cascata. Se o HTML define a estrutura de uma página, o CSS define a aparência: cores, tamanhos, espaçamentos, posicionamento, tudo que é visual.

Pensa assim: se o HTML é o esqueleto de uma casa, o CSS é a decoração — cor das paredes, tipo de piso, onde cada móvel fica. A estrutura (HTML) não muda, mas o visual (CSS) pode ser completamente diferente com o mesmo esqueleto.

A sintaxe básica sempre segue esse formato:

Box model

Todo elemento HTML é, visualmente, uma caixa retangular. Essa caixa tem quatro camadas, de dentro pra fora:

  • content — o conteúdo em si (texto, imagem)
  • padding — espaço interno, entre o conteúdo e a borda
  • border — a borda da caixa
  • margin — espaço externo, entre essa caixa e as vizinhas

06

Display

A propriedade display define como um elemento se comporta no layout. Os valores mais comuns são block (ocupa a linha toda), inline (só o espaço do conteúdo, na mesma linha) e none (some da tela completamente).

HTML
CSS
Resultado
Diferente de visibility: hidden: display: none remove o elemento completamente do layout, como se ele nem existisse. visibility: hidden esconde visualmente, mas ainda ocupa o espaço.

Position

position controla como um elemento é posicionado em relação ao fluxo normal da página ou a outros elementos.

ValorComportamento
staticPadrão — segue o fluxo normal da página
relativeDesloca em relação à posição original, sem tirar do fluxo
absoluteSai do fluxo, posiciona em relação ao pai mais próximo posicionado
fixedFica fixo na tela, mesmo com scroll

Pseudo-elementos

Diferente de pseudo-classes (que dependem de um estado), pseudo-elementos criam ou estilizam uma parte específica do elemento — como se fosse um elemento extra "fantasma", com dois-pontos duplos (::).

10

Flexbox

Flexbox é um sistema de layout feito pra organizar elementos em uma linha ou coluna, distribuindo espaço e alinhamento de forma flexível. É o jeito mais comum de alinhar coisas hoje em dia.

Pensa assim: flexbox é como organizar livros numa prateleira — você decide se ficam um do lado do outro ou empilhados, e como o espaço sobrando é distribuído entre eles.

Responsividade

Responsividade é fazer a página se adaptar a diferentes tamanhos de tela — celular, tablet, desktop. A ferramenta principal pra isso são as media queries: blocos de CSS que só se aplicam quando a tela atende a uma condição.

Media querySe aplica quando
@media (max-width: 600px)Largura da tela ≤ 600px
@media (min-width: 600px)Largura da tela ≥ 600px
@media (orientation: portrait)Tela mais alta que larga

Desafio final

Construa um pequeno cartão de produto usando flexbox. Ele deve:

  1. Usar display: flex com flex-direction: column;
  2. Ter padding, border-radius e uma sombra (box-shadow);
  3. Mudar de aparência no :hover com uma transition suave.
