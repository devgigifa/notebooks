## About

These files are personal study notes, created for learning and reviewing concepts across different programming topics.

- **HTML notebooks** are standalone files that can be opened directly in any browser.
- **Jupyter Notebooks (`.ipynb`)** require a Python environment with Jupyter installed, or can be viewed on [nbviewer](https://nbviewer.org/) or [Google Colab](https://colab.research.google.com/).
